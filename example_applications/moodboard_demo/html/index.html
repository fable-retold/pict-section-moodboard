Moodboard presentation styles

"Edit board" / "Done" toggles edit and view. In edit mode the toolbar carries the display-style toggle (canvas / jumbotron / background) and "set view area"; the gear holds the board color + backdrop margin. Pick a style, then click Done to view it.

This paragraph is host page content below the board. A jumbotron clips the board to a hero band whose height is the view-area frame scaled to the page width. A background fills the board container as a full-width backdrop (honoring the top margin), and a tall board lets the page scroll. The content fills the width of the defined view area, bleeding past its edges rather than fitting the content bounding box.
